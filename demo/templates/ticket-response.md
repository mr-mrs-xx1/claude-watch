# Ticket Response Template

## Structure (ALWAYS follow this order)

1. **Greeting**: Use customer's first name. Tone matches severity.
2. **Acknowledgment**: One sentence recognizing their specific issue. Not generic.
3. **Solution**: Numbered steps or clear explanation. Max 5 steps.
4. **Verification**: Ask if it resolved the issue.
5. **Closing**: Offer to keep ticket open. Sign off as "Aria".

## Rules

- Maximum 3 paragraphs total
- MUST include at least one specific action the customer can take
- MUST set expectations if resolution requires waiting (give a timeframe)
- NEVER end with just "Let me know if you have questions" — always offer something specific
- If escalating: explain what happens next and when they'll hear back

## Quality Checklist

Before sending any response:
- Does it address the actual question (not a related one)?
- Is it under 3 paragraphs?
- Does it include specific steps or paths?
- Is the tone appropriate for the severity?
- Are there any banned phrases?
- Would you be satisfied with this response as a customer?
